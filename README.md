# astrbot_plugin_speak_rank 插件文档

## 插件简介

`astrbot_plugin_speak_rank` 是一个用于统计群聊中成员发言情况的插件。该插件能够按天统计群内成员的文本和图片发言数量，并在设定的时间点自动生成排行榜图片发送至指定群聊。发送完成后，插件会自动清理当天统计数据，确保数据新鲜度。

本插件旨在帮助群管理员了解活跃成员分布，提升社群互动氛围。

## 功能说明

- **发言统计**：统计群聊中每个成员每日发送的文本消息和图片消息数量。
- **定时发送**：在预设时间点自动将前一天24小时内的发言排行榜以图片形式发送到白名单群。
- **自动清理**：每次发送成功后，自动清除相关统计数据，避免重复计算。
- **自定义模板**：支持自定义排行榜图片模板，包含群名、发言人数、总发言数等变量。
- **灵活配置**：可通过配置文件设置统计群白名单、定时发送时间及最多计入统计的人数。

## 插件流程

### 工作流程概述

1. **初始化阶段**
   - 加载插件配置项（如白名单群ID、定时发送时间等）。
   - 初始化数据库连接，准备存储每日发言数据。
   - 注册定时任务处理器。

2. **实时监听阶段**
   - 监听所有群聊消息事件。
   - 判断消息来源是否属于白名单群。
   - 若符合条件，则记录该消息的基本信息（发送者ID、消息类型等）至临时数据库。

3. **定时处理阶段**
   - 每天到达预设时间点时触发定时任务。
   - 查询过去24小时内各群的发言数据。
   - 计算每位成员的发言次数并排序。
   - 使用预设模板生成排行榜图片。
   - 将排行榜图片分别发送至对应的白名单群。
   - 发送完毕后清空本次统计的所有数据。

4. **异常处理与恢复**
   - 对于网络波动导致的发送失败等情况，具备一定的容错能力。
   - 日志记录关键操作过程以便排查问题。
